# Portfolio Map

## English

The portfolio is intentionally split into three repositories so that each repository can stay coherent:

- `claude-code-runtime`: runnable compatibility implementation
- `claude-code-playbook`: research, architecture notes, and onboarding docs
- `operator-kernel`: new-brand reusable agent kernel

## 中文

项目集故意拆成 3 个仓库，是为了让每个仓库只承担一种清晰职责：

- `claude-code-runtime`：可运行的兼容实现
- `claude-code-playbook`：研究材料、架构说明和导读文档
- `operator-kernel`：全新品牌的可复用 Agent 内核
